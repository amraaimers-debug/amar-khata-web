# আমার খাতা — প্রোডাকশন সেটআপ গাইড

এটা একটা Next.js + Supabase অ্যাপ। ধাপে ধাপে নিচের কাজগুলো করুন।

## ১. Supabase প্রজেক্ট বানান
1. https://supabase.com -এ লগইন করুন (আপনার অ্যাকাউন্ট তো আছেই)।
2. "New project" চাপুন, নাম দিন `amar-khata`, একটা পাসওয়ার্ড সেট করুন, region হিসেবে কাছের কোনোটা (যেমন Singapore) বেছে নিন।
3. প্রজেক্ট তৈরি হতে ১-২ মিনিট লাগবে।

## ২. ডেটাবেজ স্কিমা বসান
1. বাম পাশের মেনু থেকে **SQL Editor** এ যান।
2. **New query** চাপুন।
3. এই ফোল্ডারের `supabase-schema.sql` ফাইলের পুরো লেখা কপি-পেস্ট করে **Run** চাপুন।
4. এতে তিনটা টেবিল (posts, comments, likes) আর সব নিরাপত্তা নিয়ম (RLS) বসে যাবে।

## ৩. নিজেকে অ্যাডমিন বানান (একমাত্র লেখক আপনি)
1. বাম মেনু থেকে **Authentication -> Users** এ যান।
2. **Add user** চাপুন, নিজের ইমেইল আর একটা পাসওয়ার্ড দিন।
3. এই ইমেইল-পাসওয়ার্ড দিয়েই সাইটের `/login` পেজ থেকে লগইন করে লেখা প্রকাশ করবেন। অন্য কেউ লগইন করতে পারবে না, শুধু পড়তে/লাইক দিতে/মন্তব্য করতে পারবে।

## ৪. API কী কপি করুন
1. বাম মেনু থেকে **Project Settings -> API** এ যান।
2. `Project URL` আর `anon public` কী — এই দুটো কপি করুন।

## ৫. লোকাল কম্পিউটারে চালান
```bash
# এই ফোল্ডারে ঢুকে
npm install
cp .env.local.example .env.local
# .env.local ফাইল খুলে উপরের ধাপ ৪-এর URL আর কী বসিয়ে দিন
npm run dev
```
এবার http://localhost:3000 এ সাইট খুলবে।

## ৬. GitHub-এ কোড তুলুন
```bash
git init
git add .
git commit -m "আমার খাতা - প্রথম ভার্সন"
# GitHub-এ একটা নতুন খালি রিপোজিটরি বানিয়ে তার URL এখানে বসান
git remote add origin https://github.com/আপনার-ইউজারনেম/amar-khata-web.git
git branch -M main
git push -u origin main
```

## ৭. Vercel-এ ডিপ্লয় করুন
1. https://vercel.com -এ গিয়ে GitHub দিয়ে লগইন করুন।
2. **Add New -> Project** চেপে আপনার `amar-khata-web` রিপোজিটরি বেছে নিন।
3. **Environment Variables** সেকশনে `NEXT_PUBLIC_SUPABASE_URL` আর `NEXT_PUBLIC_SUPABASE_ANON_KEY` যোগ করুন (ধাপ ৪-এর মান দিয়ে)।
4. **Deploy** চাপুন। কয়েক মিনিটে একটা লাইভ লিংক পেয়ে যাবেন (যেমন `amar-khata-web.vercel.app`)।
5. পরে চাইলে Vercel-এর Settings থেকে নিজের ডোমেইনও (যেমন amarkhata.com) যোগ করতে পারবেন।

## এখন যা যা কাজ করে
- হোমপেজে সব লেখা দেখা যায় (Supabase থেকে সরাসরি আসে)
- প্রতিটা লেখায় ঢুকে সম্পূর্ণ পড়া, লাইক দেওয়া, মন্তব্য করা যায় — সব ডেটাবেজে স্থায়ীভাবে জমা থাকে
- শুধু আপনি `/login` দিয়ে ঢুকে `/publish` থেকে নতুন লেখা প্রকাশ করতে পারবেন

## পরের ধাপে যোগ করা যেতে পারে

- লেখা এডিট/মুছে ফেলার অ্যাডমিন প্যানেল
- ছবি আপলোড (Supabase Storage দিয়ে)
